Nice scrollbar add scrollbar as your own taste / choice.

== Description ==
One click install & active nice scrollbar plugin and get nice scrollbar to your site. You can easily customize the plugin from dashboard. It's awesome.


== Installation ==

1. Install as a regular WordPress plugin
2. After activate of this plugin Go to => Settings => Nice scrollbar and enable it
3. You can customize the scrollbar as your own taste.
